Picture Upload
===============

###About
Ruby on Rails app.

*CRUD:* users, 'photos'.

Users can upload, view, and delete, their images.  They should not be able to view others' images.

###Working online [here](http://picture-upload-with-delete.herokuapp.com/)

### Dependencies
* paperclip
* rspec
* capybara
* active record
* postgresql
* bootstrap
